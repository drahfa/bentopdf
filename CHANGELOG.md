# Changelog

All notable changes to SitiPDF Web will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.2.0] - 2026-02-05

### Added
- Light/dark mode toggle in navbar (desktop and mobile) with preference saved to localStorage
- Light mode set as default theme for new visitors
- CSS custom properties for all theme colors enabling smooth theme switching

### Changed
- Hero headline updated from "The PDF Toolkit built for privacy" to "Edit PDFs. Stay private."
- All favicon and logo references switched from SVG to PNG
- LinkedIn footer link updated to new company page
- Rebrand: all remaining "BentoPDF"/"Bentopdf" references in i18n locale files (13 languages) replaced with "SitiPDF"

### Removed
- "All Tools" link from navbar (desktop and mobile)
- "Why SitiPDF?" section from About page
- "Ready to get started?" CTA section from About page
- Digital Signature and Validate Signature tools from the tool grid
- GitHub, Discord, and Instagram links from footer (kept LinkedIn and X)

## [2.1.0]

### Changed
- Rebrand to SitiPDF and clean up UI
