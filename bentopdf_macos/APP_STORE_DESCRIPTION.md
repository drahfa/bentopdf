## Description

Transform Your PDF Workflow on macOS

SitiPDF is a powerful, native macOS PDF editor that puts professional-grade PDF tools at your fingertips. With a stunning glassmorphism interface and essential PDF tools, managing PDFs has never been easier.

PDF Annotation & Editing
- Full-featured PDF editor with real-time annotation
- Highlight text with customizable colors and opacity
- Draw freehand annotations with adjustable thickness (1-20px)
- Add shapes, stamps, and signatures
- Drag, resize, and delete annotations with ease
- Keyboard shortcuts for precise control
- Export annotated PDFs with high-quality rendering

Page Management
- Merge multiple PDFs into one document
- Split PDFs by page ranges
- Extract specific pages to new files
- Delete unwanted pages with preview
- Organize pages with drag-and-drop reordering
- Rotate pages individually or in bulk
- Automatic orientation detection (portrait, landscape, mixed)

Image Conversion
- Convert PDFs to images (PNG, JPEG)
- Create PDFs from multiple images
- Batch image processing
- High-quality output with customizable settings

Security Features
- Encrypt PDFs with password protection
- Decrypt password-protected PDFs
- Secure your sensitive documents

SitiPDF is a native macOS PDF toolkit that processes all documents locally on the user's device. No internet connection is required for core functionality.

The app uses the following capabilities:
- File access (for opening and saving PDFs)
- Local file system access (for PDF processing)
- No network requests
- No data collection
- No third-party analytics

All PDF operations are performed using the pdfx and pdf packages for Flutter, ensuring high-quality output and compatibility with standard PDF specifications.
